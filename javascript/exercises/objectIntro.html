Exercises: Objects

Object Property Access

Major US Cites: List all major cities two times, in the majorCities object, using console.log statements. The first time, list all the major cities using dot notation. The second time, list all the major cities using bracket notation. 

   var majorCities = {
      NewYork : "New York City",
      California: "Los Angeles",
      Illinois: "Chicago",
      Texas: "Houston",
      Pennsylvania: "Philadelphia",
      Arizona: "Phoenix"
   }


/* Dot Notation Solution */
console.log(majorCities.NewYork); // "New York City"
console.log(majorCities.California); // "Los Angeles"
console.log(majorCities.Illinois); // "Chicago"
console.log(majorCities.Texas); // "Houston"
console.log(majorCities.Pennsylvania); // "Philadelphia"
console.log(majorCities.Arizona); // "Phoenix"

/* Bracket Notation Solution */
console.log(majorCities["NewYork"]); // "New York City"
console.log(majorCities["California"]); // "Los Angeles"
console.log(majorCities["Illinois"]); // "Chicago"
console.log(majorCities["Texas"]); // "Houston"
console.log(majorCities["Pennsylvania"]); // "Philadelphia"
console.log(majorCities["Arizona"]); // "Phoenix"

Object Property Access

Store each city name in a separate variable. Use both bracket and dot notation. Use console.log statements to check your work. 



  var city1 = majorCities["NewYork"]);
  var city2 = majorCities["California"]);
  var city3 = majorCities["Illinois"]);
  var city4 = majorCities.Texas;
  var city5 = majorCities.Pennsylvania;
  var city6 = majorCities.Arizona;

  console.log(city1) // "New York City"
  console.log(city2); // "Los Angeles"
  console.log(city3); // "Chicago"
  console.log(city4); // "Houston"
  console.log(city5); // "Philadelphia"
  console.log(city6); // "Phoenix"

Object Property Access With Variables

In the first example, you used strings with bracket notation to access property values. In this exercise use variable names to access property values.

  var state1 = "NewYork"
  var city1 = majorCities[state1];
  console.log(city1) // "New York City"
